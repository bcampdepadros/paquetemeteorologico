# Convertir grados Celsius a Fahrenheit

Convierte una temperatura o vector de temperaturas de grados Celsius a
Fahrenheit.

## Usage

``` r
celsius_to_fahrenheit(celsius)
```

## Arguments

- celsius:

  Un valor numerico o vector numerico que representa grados Celsius.

## Value

Un valor numerico o vector numerico en grados Fahrenheit.

## Examples

``` r
celsius_to_fahrenheit(0)
#> [1] 32
celsius_to_fahrenheit(c(10, 25, 30))
#> [1] 50 77 86
```
